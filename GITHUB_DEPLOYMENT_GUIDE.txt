METHOD 1: Upload via GitHub Website (Easiest)
────────────────────────────────────────────────────────────────────────────────
1. Go to github.com and log in
2. Click the "+" icon (top right) → "New repository"
3. Name your repository (e.g., "sdg13-climate-action")
4. Choose "Public" (required for free GitHub Pages)
5. Click "Create repository"
6. Click "uploading an existing file"
7. Drag and drop these 3 files:
   • index.html
   • style.css
   • script.js
8. Scroll down and click "Commit changes"
9. Go to Settings → Pages (left sidebar)
10. Under "Source", select "Deploy from a branch"
11. Select "main" branch and "/ (root)" folder
12. Click "Save"
13. Wait 1-2 minutes, then refresh the page
14. Your site URL will appear: https://[your-username].github.io/[repo-name]/

────────────────────────────────────────────────────────────────────────────────

# Then follow steps 9-14 from Method 1
